A 3D Multiplayer Tag Game.

To run the game execute the "Tag Game" file in the build folder and select "host game". Repeat but select "join game" and you should two players in a match together.

All players must press their "ready up" button for the game to start. When this happens the players positions are reset and someone is randomly assigned as "it".

WASD - Movement

SPACE - Jump

T - Tag

C - Crouch

Left Shift - Boost (when player has boost)

Left-Control - Walk (instead on run)

Player starts with 100 boost and regenerates it slowly if they have less than 100.

Players movement and animations are synced between clients
